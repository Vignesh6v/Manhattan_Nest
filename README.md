# Manhattan_Nest

* Developed a map visualization tool for housing recommendation.
* The recommendation is based on factors like 
  * cost of living 
  * crime rate
  * nightlife
  * proximity to subways 
  * hospitals  
  * restaurants 
* used Yelp API and Open data source.
